# Skills-only plugin submission readiness

This packet is a reviewer-runnable, skills-only preflight inspired by the public plugin submission requirements. It does not submit or contact a portal. Run all eight cases against the final bundle and verify canonical/adapter parity. Keep status **NOT READY** while any gap checklist item is user-owned and unverified.

## Gap checklist

- [ ] Verified publisher identity
- [ ] Production logo
- [ ] Production category
- [ ] Website
- [ ] Support contact
- [ ] Privacy policy
- [ ] Terms of service
- [ ] Availability/region confirmation
- [ ] Release notes

Final bundle instructions: freeze the source commit, regenerate adapters, run plugin validation, compare byte parity, and attach visible outputs plus reviewer results. Do not invent missing identity, branding, legal, availability, or release materials.

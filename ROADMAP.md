# Roadmap

Diátaxis Docs is an actively developed public alpha. Priorities are evidence-driven; reproduced failures outrank speculative features.

## Alpha

- Prove Doctor in independent repositories without weakening approval or isolation boundaries.
- Dogfood Init, Doctor, context, map, and lifecycle recovery against varied disposable repositories; measure retrieval accuracy, truth preservation, privacy, and context cost before declaring a beta threshold.
- Keep the map/current-truth byte figure as provisional telemetry until dogfood evidence justifies any optimization guard.
- Run dated live compatibility smokes for Claude, Grok, Cline, Copilot, Cursor, Gemini, and OpenCode.
- Finish repository-sync and marketplace packaging only where the host provides a stable contract.
- Publish sanitized outcomes, costs when exposed, failures, and limitations—never hidden reasoning.

## Beta gate

- Successful use in at least two independent repositories.
- Green Windows and Linux CI plus public CodeQL analysis.
- Stable install and update instructions for every advertised compatibility tier.
- No regression in read-only behavior, path confinement, secret safety, dirty-worktree preservation, or approval boundaries.
- Measured command-specific adapter prompts with a regression guard backed by observed headroom, plus evidence that prompt size does not hide safety or truth semantics.

## Later

- Run the full 108-trajectory comparison only after a pilot establishes expected cost and the owner approves it.
- Consider additional Statusnone skills only when each one can meet the same evidence, safety, and maintenance bar.
